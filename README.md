# Beauty Care Website 💄✨

## Overview

The Beauty Care Website is a personalized skincare and fragrance experience designed to help users discover products based on their skin type, skin sensitivity, and scent preferences.

Users can explore beauty products with:

- Product descriptions
- Scent notes
- Product inspiration stories
- Ingredient information
- Pricing
- Personalized recommendations

The website includes a skin assessment questionnaire that recommends products based on user responses.
